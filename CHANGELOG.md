# Changelog

## 2026-03-18

### 前台兼容与样式优化
- 修复 PHP 8.2 下的多处兼容问题，包括话题描述为空、权限匹配空值、分词类动态属性等导致的页面异常。
- 修复空搜索页渲染问题，空关键词访问搜索页时不再出现空白。
- 优化移动端导航：重做折叠菜单、搜索区、登录态操作区和语言切换布局，关闭移动端搜索联想浮层，减少遮挡和错位。
- 优化移动端分页：分页区改为横向滚动，避免页码溢出视口。
- 优化文章页移动端工具栏：桌面端保留原有左侧工具条，移动端改为侧边手柄唤起的紧凑工具组，减少对阅读区域的干扰。
- 统一文章、单页、公告、问题、回答、帮助页的正文排版，补充标题、摘要、正文、代码块、表格、列表、引用等样式规范。
- 优化评论区和操作区按钮样式，统一交互视觉风格。
- 清理旧富文本残留，规范空链接、冗余属性和嵌套代码块结构。
- 优化页脚信息层级，新增隐私声明和用户协议入口，并减轻视觉重量。
- 修复文章页、隐私页、协议页、帮助页等多处移动端间距与内容容器问题。
