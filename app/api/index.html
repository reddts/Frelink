Frelink API v1

这是 Frelink 的接口入口目录。当前 API 主要服务于移动端、站点集成和 agent 辅助能力接入。

默认路由格式为 /api/{controller}/{function}，版本通过请求头 version: v1 指定。

更完整的接入说明见 docs/api-v1.md。运营洞察接口采用最近 1 / 3 / 7 / 30 天窗口，不使用累计 CTR。

公开接口

  • GET /api/Common/config
  • GET /api/Common/search?q=关键词
  • GET /api/Question/index
  • GET /api/Article/index
  • GET /api/Topic/index
  • POST /api/Insight/track

运营洞察

  • GET /api/Insight/summary?days=7
  • GET /api/Insight/keywords?days=7&limit=10
  • GET /api/Insight/opportunities?days=7&limit=10
  • GET /api/Insight/content_trends?days=7&limit=10

接入约定

  • 认证头：UserToken
  • 版本头：version: v1
  • 建议为 agent 使用独立账号与权限
  • 生产环境优先走低风险只读能力
  • 洞察数据以最近周期窗口为准
