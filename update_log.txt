【FreCenter v4.0 更新日志】
由 WeCenter V4.1.1 优化升级而来。

Bug修复
2024.11发现发表markdwon格式内容和超长文本的时候存在溢出问题和字符串解析错误问题，已经临时修复，修改涉及文件app/model/article.php
建议后续  增加使用一个专门的 Markdown 解析库
composer require erusev/parsedown
然后，使用它来将 Markdown 转换为 HTML，然后再对 HTML 使用过滤函数。
示例：
require 'vendor/autoload.php';

use Parsedown;

function sqlFilter($input) {
    // 解析 Markdown
    $parsedown = new Parsedown();
    $html = $parsedown->text($input);

    // 转换 HTML 为安全的字符串
    return strip_tags($html); // 或者其他过滤逻辑
}

$postData['message'] = sqlFilter($postData['message']);
